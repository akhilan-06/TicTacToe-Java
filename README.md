# Tic-Tac-Toe Game in Java

A console-based Tic-Tac-Toe game built using Java. This project allows two players to play Tic-Tac-Toe, detects winners, handles draw situations, and supports multiple rounds.

## Features

- Two-player gameplay
- Win detection
- Draw detection
- Input validation
- Multiple rounds support
- Console-based interface

## Technologies Used

- Java
- Arrays (2D Array)
- Loops
- Conditional Statements
- Methods

## How to Run

### Compile

```bash
javac TicTacToe.java
```

### Run

```bash
java TicTacToe
```
